# Codeblock Interpreter

Визуальный интерпретатор алгоритмов в стиле Codeblock.

## Модули
- **Модуль 1**: Основа (переменные, присваивание, арифметика, if)
- **Модуль 2**: Сортировка пузырьком (циклы, массивы, if-else)
- **Модуль 3**: HARD (функции, строки, отладка)

## Установка
1. Клонировать репозиторий
2. Открыть index.html в браузере
